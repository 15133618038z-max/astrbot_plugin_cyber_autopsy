# 赛博案底体检 — Cyber Autopsy

> 每个人在群里说过的话，都是最真实的自己。
> 让可不替你翻遍聊天记录，生成一份赛博人格画像。

---

## 🔍 这是什么

一个基于群聊历史消息的**人格分析插件**。

它会从 QQ 群中拉取目标用户的历史发言，交给 LLM 深度分析，最终生成一张精美的**赛博人格画像图**。

不是简单的关键词统计，而是真正的**性格解剖**。

---

## ✨ 功能特性

### 📊 多维度人格分析

| 维度 | 说明 |
|------|------|
| 🧠 神人值 | 衡量离谱程度，普通人 30-50，真正的神人 85+ |
| 🎭 素质水平 | 满嘴口癖 50 以下，有礼貌有内容 80+ |
| 🔮 MBTI | 基于聊天内容推断性格类型 |
| 🐾 基因兽设 | 用一种动物来形容这个人 |

### 📝 深度优缺点分析

不是冰冷的数据罗列，而是像朋友聊天一样自然地分析一个人的优缺点，引用原话作为证据，有观点、有态度。

### 🎨 精美图片输出

分析结果渲染为一张设计感十足的长图，直接发送到群里，开箱即用。

### 📜 大规模历史消息拉取

- 通过 QQ API 分页拉取群历史消息
- 自动筛选目标用户的发言
- 与内存实时缓存合并去重

---

## 🚀 使用方式

在群聊中发送：

```
画像123456789
```

将 `123456789` 替换为目标用户的 QQ 号。

Bot 会回复：

```
正在调取该用户的 500 条赛博案底，可不开始审判...
```

然后发送一张精美的分析图片。

## ⚙️ 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `llm_provider_id` | 用于分析的 LLM 提供商 ID | 空（使用当前会话模型） |
| `max_analysis_messages` | 每个用户最多分析多少条消息 | 100 |
| `protected_qq_list` | 受保护用户名单，不允许被分析 | 空 |

---

## 🔧 工作原理

```
用户发送「画像QQ号」
    ↓
分页拉取群历史消息（最多 50 页 × 200 条）
    ↓
筛选目标用户的纯文本发言
    ↓
与内存实时缓存合并去重
    ↓
截取最近 N 条消息
    ↓
LLM 深度分析人格
    ↓
渲染为精美长图
    ↓
发送到群聊
```
